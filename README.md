# 英語学習フロー｜小学生 → 中学生 → 高校生

日本で英語学習を始める小学生から、中学生・高校生までの学習の流れを1本につなげた学習計画サイトです。

🌐 **公開URL: https://english-learning-roadmap.vercel.app**
📦 リポジトリ: https://github.com/ibyichi-gif/english-learning-roadmap

## 特徴

- **小学生 → 中学生 → 高校生** の連続した学習フローを可視化
- 各段階で「ゴール」「語彙・文法の目安」「学習ステップ」「つまずきポイント」を整理
- **2つの学習パターン**を切り替え可能
  - 🧱 **具体学習（じっくり学習）**: 各ステップを丁寧に。診断→講義→暗記→演習を反復し、理解の穴を残さず積み上げる
  - ⚡ **簡易学習（短期・範囲拡大）**: 頻出・中心ルールに絞って高速周回。短期間で全体像をつかむ

## 参考

高校英語パートは [atama+ オンライン｜高校英語の学習について](https://aponline.atama.plus/posts/a000033) の「4本柱（英文法・語法／英文読解／長文演習／英単語）」の考え方を土台にしています。

## 構成

| ファイル | 役割 |
|---|---|
| `index.html` | ページ本体 |
| `styles.css` | スタイル |
| `script.js` | フロー描画・モード切替ロジック |
| `data.js` | 学習フローのカリキュラムデータ |

## ローカルで開く

静的サイトです。`index.html` をブラウザで開くだけで表示できます。
簡易サーバーで確認する場合:

```bash
npx serve .
```

## デプロイ

Vercel にそのままデプロイ可能（ビルド不要の静的サイト）。
