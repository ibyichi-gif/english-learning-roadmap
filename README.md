# 英語習熟トレーナー｜小学生・中学生・高校生

日本で英語学習を始める小学生から、中学生・高校生までの英語を **「区分 × 学習単元」** で習熟していく学習アプリです。

🌐 **公開URL: https://english-learning-roadmap.vercel.app**
📦 リポジトリ: https://github.com/ibyichi-gif/english-learning-roadmap

## 特徴

- **3つの学習区分**（小学生 / 中学生 / 高校生）に分け、それぞれを複数の **学習単元** に細分化
- 各単元は **「診断 → 講義 → 演習」** のステップで習熟を進める
- 演習は4つの問題形式に対応
  - 選択式 / 並べ替え式 / 入力式 / 品詞識別
- 演習の正答率に応じて **習熟レベル**（未学習 → 初級 → 中級 → 上級 → 合格）が変化
- 全問正解で **メダル**（1周＝🥈銀 / 2周＝🥇金 / 3周＝🌟シャイニー）を獲得
- 間違えた問題は **「苦手 / うろ覚え / 覚えた」** の記憶度として記録され、「習熟状況」画面から復習可能
- 前回学習から1日以上たった単元は **「復習推奨」** として習熟状況に表示
- **「学習のコツ」ページ** に、英単語・英文法・長文読解・リスニングの分野別の進め方を収録
- 2つの学習モード
  - 🧱 **じっくり学習**: 診断 → 講義 → 演習をフルで進める王道ルート
  - ⚡ **短期学習**: 講義は要点のみ、演習中心で高速周回する総復習ルート
- 学習データはブラウザの localStorage に保存（サーバー不要）

## 学習単元（全29単元・演習130問）

| 区分 | 単元 |
|---|---|
| 小学生 | アルファベットとフォニックス／身の回りの基本単語／be動詞の文／一般動詞の文／疑問文と答え方／あいさつと自己紹介 |
| 中学生 | be動詞と一般動詞／三単現のs／過去形／現在進行形／未来表現／助動詞／現在完了／比較／受動態／不定詞と動名詞／関係代名詞／頻出英単語（中3レベル） |
| 高校生 | 5文型／時制の整理／助動詞＋have＋過去分詞／仮定法／関係詞／分詞と分詞構文／英文読解①〜③／英単語（入試レベル）／頻出英単語（高2・高3レベル） |

## 構成

| ファイル | 役割 |
|---|---|
| `index.html` | アプリのシェル |
| `styles.css` | スタイル |

## ローカルで開く

静的サイトです。`index.html` をブラウザで開くだけで動作します。

## デプロイ

ビルド不要の静的サイトとして Vercel にデプロイしています。
