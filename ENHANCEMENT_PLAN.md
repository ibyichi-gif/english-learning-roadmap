# 英語習熟トレーナー 強化計画（現状40% → 目標120%）

学習の目的：**内容の「把握」と「定着」**。
現状（40%）に対し、把握＝理解を助ける工夫、定着＝忘れさせない工夫を、各単元に最適な形で重ねる。

## 現状（40%）でできていること
診断→講義→演習／4+2の問題形式／構造化解説（模範解説・ポイント・詳しい解説）／
習熟レベル・メダル／わからないボタン／経過タイマー／苦手・うろ覚え記録／復習推奨フラグ

## 強化の柱（120%へ）

| # | 機能 | 目的 | 適用範囲 |
|---|---|---|---|
| F1 | 小学生の学年対応（小1〜から学べるロジック） | 把握 | 小学生区分（＋全単元に学年目安） |
| F2 | 音声読み上げ（Web Speech API / TTS） | 把握・定着 | 英文・英単語・長文を含む全単元 |
| F3 | 段階ヒント（答える前に手順の入口を提示） | 把握 | 全演習（ポイントの第1手順をヒント化） |
| F4 | 本格SRS（間隔反復）＋復習センター | 定着 | 全問題（翌日→3日→7日→16日→35日） |
| F5 | 間違いだけ再挑戦 | 定着 | 演習結果画面 |
| F6 | 学習ストリーク（連続学習日数） | 定着（習慣化） | アプリ全体 |

---

## F1 小学生の学年対応（ロジック）
- 全単元に **学年目安（gradeFrom）** を持たせ、バッジ表示。
  - 小学生：小1〜（フォニックス／基本単語／あいさつ）、小3〜（be動詞／一般動詞）、小4〜（疑問文）
  - 中学生：中1〜／中2〜／中3〜　高校生：高1〜／高2〜／高3〜
- **区分ページは学年目安順に並べ替え**。小1の学習者は「いちばん上の単元」から無理なく始められる。
- 小学生区分に「**小学1年生からでも、上の単元から順に始められます**」の案内を明示。
- これにより「どの学年からでも、自分の位置から積み上げる」導線がロジックとして組み込まれる。

## F2 音声読み上げ（TTS）
- ブラウザの音声合成で英語を読み上げる 🔊 ボタンを、英文・例文・長文・並べ替え解答に付与。
- 小学生＝フォニックス／基本単語で「音と文字の一致」、中高＝例文・長文で「音読・リスニングの素地」。
- 音声アセット不要・無料でエレガントに「把握」を底上げ。

## F3 段階ヒント
- 各問題に「ヒント」ボタン。`hint` 指定があればそれ、なければ **解説のポイント第1手順**を提示。
- 減点はしないが「ヒント使用」を記録。考える入口を与え「把握」を助ける。

## F4 本格SRS（間隔反復）＋復習センター
- 問題ごとに記憶度に応じた **次回復習日** を設定（正解で間隔を延ばし、不正解で翌日に戻す）。
- 新ページ「**復習センター**」：
  - **今日の復習** … 復習日が来た問題をまとめて出題（区分をまたいで）。
  - **弱点をまとめて復習** … 苦手・うろ覚えだけの復習セッション。
- atama+ の「うろ覚え／苦手を高速回転」「翌日→3日後→1週間後」をアプリのコア機能に。

## F5 間違いだけ再挑戦
- 演習結果画面に「間違えた問題だけやり直す」。その場で弱点だけ詰め直せる。

## F6 学習ストリーク
- 連続学習日数をトップバーに表示。毎日少しずつ続ける動機づけ（定着の習慣化）。

## 区分・単元ごとの最適適用方針

### 小学生区分（学年：小1〜小4目安）
- **アルファベットとフォニックス（小1〜）**：F2音声が主役。文字→音の一致を耳で確認。
- **身の回りの基本単語（小1〜）**：F2音声＋F4 SRSで語彙を反復定着。
- **あいさつと自己紹介（小1〜）**：F2音声で定型文を音から覚える。
- **be動詞／一般動詞の文（小3〜）**：F3ヒントで「型」の入口を提示。
- **疑問文と答え方（小4〜）**：F3ヒント＋F2音声でやり取りを音で。

### 中学生区分（学年：中1〜中3目安）
- 文法11単元＋頻出英単語：F3ヒント（ポイント第1手順）で語順ルールの入口を提示。
- 並べ替え問題：F2音声で完成文を読み上げ、語順を耳で確認。
- 頻出英単語（中3）：F4 SRSで「翌日→3日→7日」反復、F2音声で発音同時。

### 高校生区分（学年：高1〜高3目安）
- 文法・語法：F3ヒント＋構造化解説で「解き方の手順」を徹底。
- 英文読解①〜③（steps型）：F2音声で英文・長文を読み上げ、音読の素地に。
- 品詞識別（label型）：F3ヒントで「まず動詞を探す」など着眼点を提示。
- 頻出英単語（高2・高3）：F4 SRSで入試語彙を長期記憶化、F2音声併用。

### 横断
- F4復習センター・F5間違い直し・F6ストリークは全区分共通で「定着」を支える。
